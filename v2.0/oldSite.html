The Club Site

In version 1.0 of our site, the entire site was made using only raw CSS and HTML. By starting with the fundamentals, we were able to understand the core aspects of what it takes to make a website. Although the site was very rudimentary, it was a good springboard into the world of web development.
In version 2.0, we completely rebuilt the site from the ground up, only this time we used Twitter' Bootstrap 3.0 library to help with page layout and adding new features. By using Bootstrap, we were able to build the site much quicker than using raw HTML and CSS. Also added in version 2.0, was the use of Javascript and the popular jQuery library to help with website animation.
