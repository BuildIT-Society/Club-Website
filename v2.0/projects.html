Building the Future

Project work is the base foundation of our club. It is what brings us together as a society of creators. We have structured the club so that each semester we will work on a least one project and try to complete it by the end of the semester. At the end of each semester, Purdue hosts a engineering challenge called the Spark Challenge, where students from many different engineering displines build projects using micro-processors or micro controllers. In the past, we have made projects that can compete in the Spark Challenge, but we are not limited to projects of this catagory alone. If you are interested in the Spark Challenge, you can learn more here.
All project files and code can be found at our GitHub here.
