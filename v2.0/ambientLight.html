Arduino Microcontroller

For this project, we decided to use an Arduino Uno 3 to run our program on. Arduino's are great for creators looking to rapidly prototype new technologies and iterate project updates on. As a mircrocontroller, Arduino's simply run code through a loop as long as they retain power, so this was a perfect fit for our ambient lighting system.

RGB Light Strip

The main component of our project was the LED RGB light strip. The RGB strip allowed use to program the light color using values from 0-255 for red, green, and blue. Using this capability, we designed the system to have 4 main modes: average color pixel from laptop, changing color with music, joystick color picker, and rainbow mode.
The average color pixel mode used a 3rd party software called ScreenShoter to collect the current average pixel color of the user's screen. Once we had the color, we then displayed it on the light strip.
The changing lights from music was created by randomly cycling through colors and changing colors on the beat of music. We used a microphone to collect audio samples and then created a queue that stored audio snippet amplitudes in order to find the average amplitude of the song. Once we had the average amplitude, we changed the LED color if the dequeued sound snippet was greater than the average.
The rainbow mode used a circular linked-list to scroll through a cascade of colors to be displayed on the lightstrip. The modes continued until a button was pressed on the lighting system at which point it changed to the next light mode.

Coloring Choosing Mode

The joystick color picker mode offered us a unique problem. The joystick gave us a X and Y cartesian point and we need to dynamically change the light color given the position of the joystick.
