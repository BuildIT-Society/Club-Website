Capital One Globe Challenge

By Sam Bartek

What the Globe Data Represents

The globe is a data visualization of temperature data
from all the major cities around the world.
On the globe, the shorter and more blue bars represent cold temperatures,
while the taller and more red bars represent hotter temperatures.
I used data that I collected through the OpenWeatherMap API
on 10/29/2016 to be displayed on the globe as seen below.

View the Globe Here

How It Was Created

The Capital One Globe Challenge was a challenge
given to students to create a data visualization
using Google Chrome's WebGL Globe program.
For my globe, I used OpenWeatherMap's API to get
the latitude and longitude coordinates of
all the major cities in the world.
Then I retrieved the temperature data of those cities
using a python script. I then converted the
data from the python format back to JSON
since the WebGL Globe requires JSON
data to be in a unique form. After that I
sent the data to the WebGL Globe code,
which in turn displayed the geolocation data on the 3-D globe.
